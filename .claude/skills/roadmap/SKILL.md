## Step 1: Fetch Data from Linear

### For Snapshot and Update modes

Fetch all project issues grouped by roadmap label:

```
linear.list_issues(
  team: "624a9948-a160-4e47-aba5-7f9404d23506",
  project: "227427af-6844-484d-bb4a-dedeffc68825",
  first: 250
)
```

Group issues by `roadmap:*` labels. For each theme, compute:
- `total`: count of issues with this label
- `done`: count of completed issues (state = "Done" or "Cancelled")
- `in_progress`: count of in-progress issues
- `percentage`: `done / total * 100`

Issues without a `roadmap:*` label count toward "Security & Compliance" (default bucket).

**Note**: Theme percentages require `roadmap:*` label coverage on tickets. Low coverage = inaccurate percentages. Report coverage: `labeled / total * 100`.

### For Milestones mode

```
linear.list_milestones(
  teamId: "624a9948-a160-4e47-aba5-7f9404d23506"
)
```

If no milestones exist yet, display the planned milestones from the table above with a note: "Milestones not yet created on Linear. Use `/roadmap --update` to create them."
